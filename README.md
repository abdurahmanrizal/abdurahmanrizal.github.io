# Abdurahman Argoebie Portfolio

Responsive personal portfolio built with React, TypeScript, Vite, and Bun.

## Run with Bun

```bash
bun install
bun run dev
```

## Production build

```bash
bun run build
bun run preview
```

Main content is in `src/App.tsx`, styling is in `src/styles.css`, and the downloadable CV is stored in `public/Abdurahman-Argoebie-CV.pdf`.
